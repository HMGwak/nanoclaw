# original_source

Preserved source modules live here.

Each module keeps the original repo/docs/pipeline assets stable so agents, toolsets,
and flows can be derived from them without losing the source of truth.

Rules:

- Keep this directory read-mostly.
- Store original material under `source/` and `docs/`.
- Record provenance in `manifest.json`.
- Do not place service-specific bindings or deployed bot behavior here.
